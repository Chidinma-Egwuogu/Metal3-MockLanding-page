About Metal³

Welcome to Metal³, the open-source project that brings Kubernetes and bare metal infrastructure together in perfect harmony. We're all about making your life easier, one server at a time.

Our mission? To simplify bare metal infrastructure management and make Kubernetes accessible to everyone, regardless of hardware. We're committed to offering a seamless experience with a unified API, scalability, reliability, and customization options.

Metal³ is powered by three core components:

  • Bare Metal Host Operator (BMH): The behind-the-scenes Kubernetes controller working with Ironic to manage resources like a boss.
  • Cluster API Metal³ Provider: The trusty sidekick ensuring your Kubernetes clusters run smoothly on bare metal hosts.
  • Ironic Image: The open-source provisioning tool enabling Metal³ to work its magic on any kind of hardware.

Our project offers a plethora of benefits, including simplified and scalable bare-metal infrastructure management, Kubernetes goodness for all, and a rich set of APIs and tools for your specific needs.

Why Metal³ ?

Kubernetes Native API

Manage your underlying infrastructure through Kubernetes APIs using the Metal³ and machine-api CRDs.

Self-Hosted

Metal³ operates directly within your Kubernetes cluster, integrating seamlessly with your existing infrastructure, eliminating the need for external dependencies.

Self-Managed

Integration with the Machine API from SIG Cluster-Lifecycle allows your bare metal cluster to manage its own growth, just like your cloud-based clusters
